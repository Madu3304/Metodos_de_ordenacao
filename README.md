#O desenvolvimento foi realizado na linguagem Python. 

Foram implementados e comparados os desempenhos dos algoritmos Bubble Sort, Insertion Sort e Merge Sort.

A estrutura do código está organizada da seguinte forma: há três módulos, cada um responsável pela implementação de um método de ordenação; um módulo principal (main), responsável pela execução e integração do sistema; um módulo destinado à geração de dados aleatórios; e um arquivo no formato JSON, utilizado para armazenar os dados gerados a cada execução.
